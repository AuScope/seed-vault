==============
FDSN Resources
==============

Station Web Service
===================

`AusPass Station URL Builder <http://www.auspass.edu.au/fdsnws/station/1/builder>`_ can be used to search for SEISMIC STATIONS

`FDSN Station specification <https://www.fdsn.org/webservices/fdsnws-station-1.1.pdf>`_

`FDSN Station issue and suggestion tracking <https://github.com/FDSN/fdsnws-station>`_

Event Web Service
=================

`AusPass Event URL Builder <http://www.auspass.edu.au/fdsnws/event/1/builder>`_ can be used to search for EARTHQUAKE EVENTS

`FDSN Event specification <https://www.fdsn.org/webservices/fdsnws-event-1.2.pdf>`_

`FDSN Event issue and suggestion tracking <https://github.com/FDSN/fdsnws-event>`_

DataSelect Web Service
======================

`AusPass DataSelect URL Builder <http://www.auspass.edu.au/fdsnws/dataselect/1/builder>`_ can be used to search for WAVEFORM DATA

`FDSN DataSelect Specification <https://www.fdsn.org/webservices/fdsnws-dataselect-1.1.pdf>`_

`FDSN DataSelect issue and suggestion tracking <https://github.com/FDSN/fdsnws-dataselect>`_
